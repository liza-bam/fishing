Round icon-only button — carousel arrows, play, share. `glass` variant floats over photography.

```jsx
<IconButton variant="glass" label="Next" icon={<i data-lucide="arrow-right"></i>} />
<IconButton variant="outline" size="sm" label="Share" icon={<i data-lucide="share"></i>} />
```

- `variant`: `solid` (deep-sea), `outline`, `glass` (blurred, for over imagery).
- Always pass `label` for accessibility.
